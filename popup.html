Setup Guide

Step 1 — Get a Free AI Key

Choose a provider below and grab a free key:

  • Groq (Best — Free forever)
    console.groq.com → Sign up with Google → Create API Key → Copy it
  • Gemini (Google — Free)
    aistudio.google.com/app/apikey → Sign in → Create API Key → Copy it

→ Paste the key in AI Provider → API Key field above and hit Save API Key.

Step 2 — Connect GitHub

This lets the extension auto-save your solutions to a GitHub repo.

  1. Go to github.com/settings/developers
  2. Click OAuth Apps → New OAuth App
  3. Fill in:
    • Application name: Hint Engine
    • Homepage URL: https://github.com
    • Authorization callback URL: copy the Callback URL shown in the GitHub section below ↓
  4. Click Register application
  5. Copy the Client ID shown on the next page
  6. Click Generate a new client secret → copy it (shown only once!)

→ Paste Client ID in the GitHub section → click Connect GitHub → paste Client Secret when prompted.

Step 3 — Pick a Repo

After connecting, a dropdown will appear with all your GitHub repos. Select one (e.g. my-solutions) and click Save Repository.

That's it! Every accepted submission will automatically create a file like:
solutions/leetcode/two-sum.md
